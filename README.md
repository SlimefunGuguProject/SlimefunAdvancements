# SlimefunAdvancements 粘液科技进度

这是一个 Slimefun 附属，可以给 Slimefun 添加一个可配置的进度系统。

需要 Java 16+（Paper 1.21.1 构建要求），请确认服务端 JVM 版本满足要求。

注意：Vanilla 进度注入逻辑需在主线程调用。
